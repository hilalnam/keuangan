# DompetKu

A finance app monorepo with frontend and backend.

## Deploy options

### Free deployment
- Frontend: Vercel
- Backend: Render
- Database: Neon

### Required environment variables
Backend:
- DATABASE_URL
- BETTER_AUTH_SECRET
- BETTER_AUTH_URL
- FRONTEND_URL
- PORT
- ALLOW_ANONYMOUS
- DEV_USER_ID

Frontend:
- VITE_API_URL
